# @repo/web

## Prerequisites

This project uses Vercel for development and deployment. You'll need to install the Vercel CLI:

```bash
pnpm add -g vercel
```

## Development

Run the development server:

```bash
vercel dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Deployment

Deploy to Vercel:

```bash
vercel
```

For production:

```bash
vercel --prod
```
